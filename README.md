# Cybersecurity Optimization Model

A Monte Carlo simulation and optimization framework for
cybersecurity control allocation across a layered IoT network.

## Structure
- **Block 1**: Monte Carlo breach probability simulation
- **Block 2**: Summary statistics and visualisation
- **Block 3**: Log-normal error modelling
- **Block 4**: Piecewise linear coefficient fitting
- **Block 5**: Deterministic Gurobi optimization
- **Block 6**: Stochastic optimization (SAA method)
- **Block 7**: Sensitivity analysis and comparison plots

## Requirements
pip install -r requirements.txt

Note: gurobipy requires a valid Gurobi license.

## Usage
python optimization_file.py
